## 项目目标

跑通首版核心闭环：

用户输入内容 -> AI 生成题目 -> 用户答题 -> AI 生成分析报告。

技术栈：

- 前端：Taro + React 微信小程序
- 后端：Spring Boot 3 + MyBatis-Plus
- 数据库：MySQL
- AI：Spring AI Alibaba + 通义千问
